# Battery-Management-System-Final-Project
BMS Python script that uses the INA226 module and the Waveshare 2-inch LCD.

Library for ina226 module:
https://github.com/e71828/pi_ina226

Follow this guide for this specific LCD screen:
https://www.waveshare.com/wiki/2inch_LCD_Module

The wave_lib library from the link above will be provided

Parts:
-Xinimark Ina226 Bi-directional Alert Monitor module (https://www.amazon.com/dp/B07T597W1J?ref=ppx_yo2ov_dt_b_fed_asin_title)

-Seloky LM2596 DC TO DC Buck Converter (https://www.amazon.com/dp/B0DBVYP91F?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)

-Waveshare 2-inch LCD Screen SPI interface (https://www.amazon.com/dp/B082GFTZQD?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)

-Breadboard Jumpers (https://www.amazon.com/dp/B0BRTHR2RL?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)

-Raspberry pi Zero 2 w (https://www.amazon.com/dp/B0CG99MR5W?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1)
